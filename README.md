PIL01_Isaac

Este programa fue hecho con Python y distintos módulos:

PIL  
Carga y guardado: Abre y guarda imágenes en múltiples formatos con Image.open() y Image.save().
Transformaciones: Redimensiona (resize), recorta (crop), rota (rotate), y convierte a escala de grises (convert('L')).
Manipulación: Permite pegar y fusionar imágenes, manejar transparencia y aplicar máscaras.
Mejora y filtros: Ajusta brillo, contraste, aplica desenfoques, detección de bordes y nitidez.
Texto: Añade texto a las imágen
